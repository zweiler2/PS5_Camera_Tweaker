# PS5 Camera Tweaker

## Usage

### 1. Download the latest version of the PS5 Camera Tweaker:

[![Latest Release](https://img.shields.io/github/v/release/zweiler2/PS5_Camera_Tweaker?label=release&style=flat-square)](https://github.com/zweiler2/PS5_Camera_Tweaker/releases/latest)

### 2. Run the PS5 Camera Tweaker executable file and follow the onscreen instructions

2.1. Windows:

Open a command prompt or terminal and navigate to the extracted folder.
(Hint: Inside the folder where you saved the PS5 Camera Tweaker executable, click on the address bar at the top write "cmd" and press enter or shift+right-click and select "Open In Terminal")

```bat
PS5_Camera_Tweaker.exe firmware.bin
```

2.2. Linux:

Open a command prompt or terminal and navigate to the extracted directory.
(Hint: Inside directory where you saved the PS5 Camera Tweaker executable, right-click and select "Open Command Prompt Here")

```bash
./PS5_Camera_Tweaker firmware.bin
```

### 3. The resulting firmware file will be saved as `output.bin`.

Note that the firmware file you provided will not be modified.
